# 🏕️ Camp List

Spring family camping checklist with packing order.

**Features:** 4-state checkboxes (to get → buy → gathered → packed), auto shopping list with copy, long-press state picker, sort, pack all, inline editing, add/remove items & sections.

**Usage:** Open `index.html` in a browser or visit the GitHub Pages URL. Add to your phone's home screen for an app-like experience with persistent data via localStorage.
